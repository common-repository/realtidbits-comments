== Description ==

The RealTidbits comment system replaces your WordPress comment system with comments powered by RealTidbits.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click 'Settings' and enter your RealTidbits account credentials
